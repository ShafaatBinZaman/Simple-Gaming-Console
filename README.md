# Simple-Gaming-Console 🎮

A Simple Gaming Console software program enables players to play numerous games, track and view their scores, and earn achievements.This gaming platform is beneficial as users do not have to search for other games; instead, our program has its own package of games through which users can switch to any game they want without even needing to leave the platform.  

Our program enables users to feel organized and secure while saving time. Consequently, this gaming console also offers login options, a score tracker, a segment of achievements, and guidance throughout the games.

Games

🎮 Rock Paper Scissors Game
1. Game Logic:            
The game logic determines the winner of each round based on the
choices made by the user and the computer. It follows the rules mentioned :
• Rock crushes Scissors (Rock wins against Scissors).
• Scissors cut Paper (Scissors win against Paper).
• Paper covers Rock (Paper wins against Rock)
• If both players choose the same option, the round is a tie.         
2. Computer Choice:        
The computer generates a random choice (Rock, Paper, or Scissors)
for each round. This simulates the unpredictability of a human opponent.         
3. Score Tracking:             
The game keeps track of the scores for both the user and the
computer. This can be viewed from the scoreboard option in the OPTIONS panel.           
4. Displaying Results:            
The results of each round, including the choices made by the user          
and the computer and the winner of the game, are displayed to the user.                 
6. Exit Game:          
The user can keep playing other rounds or exit the game

🎮  Number Guessing Game

A Number Guessing Game is a simple game where the computer randomly selects a number within a specified range, and the player tries to guess that number within a limited number of guesses. The game typically provides feedback to the player after each guess, indicating whether the guessed number is too high, too low, or correct. The objective is for the player to think of the correct number with the fewest attempts possible.
